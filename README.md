# 🚀 AutoCare – Bootstrap 5 UI Exploration & Page Design

## 🎯 Objective

This project was developed as part of the **Bootstrap 5 UI Exploration Internship Task**.
The goal was to explore Bootstrap components and design clean, modern, and responsive web pages by combining them creatively.

As mentioned in the task instructions :

* Understand Bootstrap structure and usage
* Use components like Navbar, Cards, Forms, Hero sections
* Build visually appealing and responsive pages
* Avoid direct copy-paste — focus on remixing components
